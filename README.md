# Library Management System

Backend API, Designed Using API First Approach (Contract First).

### Stack
- Java 17
- Spring Boot 2.7.0
- PostgreSQL

### Covered
- [x] Rest API
- [x] Spring Data JPA
- [x] Objects Mapping (MapStruct)
- [x] Database Migration (Liquibase)
- [x] Unit and Integration tests 
- [x] Docker and Docker Compose
- [ ] DTO Validators
- [ ] Security
- [ ] CI
